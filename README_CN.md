# Claude 快速入门

Claude 快速入门是一系列项目集合，旨在帮助开发者快速上手使用 Claude API 构建应用程序。每个快速入门项目都提供了一个基础框架，你可以轻松地在此基础上进行扩展和定制，以满足你的特定需求。

## 开始使用

要使用这些快速入门项目，你需要一个 Claude API 密钥。如果你还没有，可以在 [console.anthropic.com](https://console.anthropic.com) 免费注册获取。

## 可用的快速入门项目

### 客户支持代理

一个由 Claude 驱动的客户支持代理。该项目展示了如何利用 Claude 的自然语言理解和生成能力，创建一个具有知识库访问功能的 AI 辅助客户支持系统。

[前往客户支持代理快速入门](./customer-support-agent)

### 金融数据分析师

一个由 Claude 驱动的金融数据分析师。该项目展示了如何利用 Claude 的能力与交互式数据可视化相结合，通过聊天方式分析金融数据。

[前往金融数据分析师快速入门](./financial-data-analyst)

### 计算机使用演示

一个 Claude 可以用来控制桌面计算机的环境和工具集。该项目展示了如何利用 Claude 的计算机使用能力，包括支持最新的 `computer_use_20251124` 工具版本及其缩放操作功能。

[前往计算机使用演示快速入门](./computer-use-demo)

### 自主编程代理

一个由 Claude Agent SDK 驱动的自主编程代理。该项目展示了一种双代理模式（初始化代理 + 编程代理），可以在多个会话中构建完整的应用程序，进度通过 git 持久化，代理会逐步完成功能列表中的各项任务。

[前往自主编程代理快速入门](./autonomous-coding)

## 通用使用方法

每个快速入门项目都附带了自己的 README 和设置说明。通常，你需要按照以下步骤操作：

1. 克隆此仓库
2. 进入特定的快速入门项目目录
3. 安装所需的依赖项
4. 将你的 Claude API 密钥设置为环境变量
5. 运行快速入门应用程序
