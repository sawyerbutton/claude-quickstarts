# Claude 金融数据分析师

![封面](public/hero.png)

一个复杂的 Next.js 应用程序，结合了 Claude 的能力与交互式数据可视化，通过聊天方式分析金融数据。

## 功能特性

- **智能数据分析**：由 Claude 驱动（使用 Claude 3 Haiku 和 Claude 3.5 Sonnet）
- **多格式文件上传支持**：
  - 文本/代码文件（.txt、.md、.html、.py、.csv 等）
  - PDF 文档（带文本的常规 PDF，不支持扫描文档）
  - 图像
- **交互式数据可视化**：根据分析的上下文和数据，Claude 可以生成以下图表：
  - 折线图（时间序列数据和趋势）
  - 柱状图（单一指标比较）
  - 多柱状图（多指标比较）
  - 面积图（随时间变化的数量/体积）
  - 堆叠面积图（组件分解）
  - 饼图（分布分析）

## 开始使用

### 前提条件

- 已安装 Node.js 18+
- Claude API 密钥（用于 Claude 集成）

### 安装

1. 克隆仓库：
```bash
git clone https://github.com/anthropics/anthropic-quickstarts.git
cd anthropic-quickstarts/financial-data-analyst
```

2. 安装依赖项：
```bash
npm install
```

3. 在根目录创建 `.env.local` 文件：
```env
ANTHROPIC_API_KEY=your_api_key_here
```

4. 运行开发服务器：
```bash
npm run dev
```

在浏览器中打开 [http://localhost:3000](http://localhost:3000) 查看结果。

## 技术栈

- **前端**：
  - Next.js 14
  - React
  - TailwindCSS
  - Shadcn/ui 组件
  - Recharts（用于数据可视化）
  - PDF.js（用于 PDF 处理）

- **后端**：
  - Next.js API 路由
  - Edge Runtime
  - Anthropic SDK

## 使用示例

该助手可以帮助完成各种金融分析任务：

1. **数据提取与分析**：
   - 上传金融文档
   - 提取关键指标
   - 分析趋势和模式

2. **可视化创建**：
   - 根据数据生成图表
   - 自定义可视化
   - 比较多个指标

3. **交互式分析**：
   - 询问有关数据的问题
   - 请求特定的可视化
   - 获取详细解释

## 有趣的使用场景

虽然主要设计用于金融分析，但 AI 助手可以适用于各种有趣的应用：

1. **环境数据分析**：
   - 分析气候变化趋势
   - 可视化污染水平随时间的变化
   - 比较不同地区的可再生能源采用情况

2. **体育表现跟踪**：
   - 上传运动员表现数据
   - 生成关键指标的可视化
   - 分析团队统计数据的趋势和模式

3. **社交媒体分析**：
   - 处理来自各种平台的互动数据
   - 创建显示粉丝增长和互动率的图表
   - 分析用户评论中的情绪趋势

4. **教育进度跟踪**：
   - 上传学生表现数据
   - 可视化学习进度随时间的变化
   - 比较不同的教学方法或课程

5. **健康和健身监测**：
   - 处理来自可穿戴设备的个人健康数据
   - 创建步数、心率和睡眠模式等指标的图表
   - 分析长期健康趋势并提供见解

你甚至可以使用图表和图像创建有趣的结果，比如使用饼图查看图片中最常见的内容。
